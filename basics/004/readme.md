# минимум двух чисел

Найдите минимальное из двух заданных чисел.

## входные данные

В первой строке входных данных даны два целых числа _A_ и _B_, разделенных пробелами.

## выходные данные

Целое число, равное _min(A, B)_.

## ограничения

 * - 1000 <= A, B <= 1000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| -- 10 15 -- | -- 10 -- |
| -- 10 -15 -- | -- -15 -- |
