# Возведение в степень

По данному действительному числу __a__ и натуральному __n__ вычислите величину $a^n$.

## входные данные

В первой строке даны действительное число __a__ и натуральное число __n__, разделенные пробелом.

## выходные данные

Вывести значение $a^n$.

## ограничения

 * -100 <= a <= 100
 * 0 <= n <= 100
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 2 10 | 1024 |
| 1.5 3 | 3.375 |
| 3 0 | 1 |
