# Сумма треугольных чисел

По данному натуральному __n__ вычислите сумму $1+(1+2)+(1+2+3)+...+(1+2+...+n)$.

## входные данные

В первой строке дано натуральное число __n__.

## выходные данные

Вывести значение суммы треугольных чисел.

## ограничения

 * 1 <= n <= 1 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3 | 10 |
| 5 | 35 |
| 1 | 1 |
