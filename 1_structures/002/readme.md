# Карточная игра

Два игрока играют в следующую карточную игру:

* В начале игры у каждого игрока есть набор из n карт, пронумерованных от 1 до n.
* Игроки ходят по очереди, начиная с первого игрока.
* Каждым ходом оба игрока выбирают верхнюю карту из своей колоды и кладут её на стол.
* Игрок, у которого карта с большим номером, забирает обе карты и кладёт их в низ своей колоды (сначала карту соперника, затем свою карту).
* Если у обоих игроков карты с одинаковым номером, то они просто сбрасываются.
* Карта с номером 1 бьет карту с номером n (то есть 1 считается больше n).
* Игра продолжается K ходов или до тех пор, пока у одного из игроков не останется карт.
* Победителем считается игрок, у которого в конце игры больше карт. Если у обоих игроков одинаковое количество карт, объявляется ничья.

определите, кто выиграет при заданных начальных колодах и числе ходов K и насколько карт больше у победителя, либо сообщите, что игра закончится вничью.

## входные данные

Первая строка содержит два натуральных числа n и K (1 ≤ n ≤ 100, 1 ≤ K ≤ 10^4) — количество карт у каждого игрока в начале игры и максимальное количество ходов.

Следующие две строки содержят по n натуральных чисел от 1 до n — начальные колоды первого и второго игрока соответственно. Числа в каждой строке разделены пробелами. Карты перечислены сверху вниз.

## выходные данные

В одной строке выведите "1 X", если выиграет первый игрок, "2 X", если выиграет второй игрок, или "0 0", если игра закончится вничью. Здесь X — разница в количестве карт между победителем и проигравшим.

## ограничения

* 1 <= n <= 100
* 1 <= K <= 10^4
* time limit: 1s
* memory limit: 8m
