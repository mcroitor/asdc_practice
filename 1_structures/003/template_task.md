# Проверка скобок

Дана строка, состоящая из символов '(', ')', '{', '}', '[' и ']'. Определите, является ли входная строка корректной с точки зрения расстановки скобок. Строка считается корректной, если:

1. Каждая открывающая скобка имеет соответствующую закрывающую скобку того же типа.
2. Скобки правильно вложены друг в друга.
3. Пустая строка считается корректной.

## входные данные

В единственной строке дана строка символов, состоящая из '(', ')', '{', '}', '[' и ']'.

## выходные данные

Выведите "YES", если строка корректна, и "NO" в противном случае.

## ограничения

* Входная строка содержит не более 1000 символов.
* time limit: 1s
* memory limit: 8m

## пример

| INPUT              | OUTPUT              |
| ------------------ | ------------------- |
| -- input sample -- | -- output sample -- |
| ()[]{}             | YES                 |
| ([)]               | NO                  |
| {[]}               | YES                 |
|                    | YES                 |
| (                  | NO                  |
