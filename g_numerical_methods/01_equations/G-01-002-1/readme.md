# Приближение числа Пи

По данному числу __n__ вычислите сумму $4(1-\frac{1}{3}+\frac{1}{5}-\frac{1}{7}+...+\frac{(-1)^n}{2n+1})$.

## входные данные

В первой строке дано натуральное число __n__.

## выходные данные

Вывести значение суммы с точностью до 6 знаков после запятой.

## ограничения

 * 0 <= n <= 100 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 0 | 4.000000 |
| 10 | 3.232316 |
| 1000 | 3.142594 |

## примечание

При увеличении __n__ эта сумма приближается к числу $\pi \approx 3.141593$ (ряд Лейбница).
