# Функция RIGHT

Дана строка _S_. Выведите последние _N_ символов данной строки.

## входные данные

В первой строке дана строка символов. Во второй строке дано натуральное число _N_.

## выходные данные

В одной строке вывести _N_ последних символов строки _S_. Если _N_ больше длины строки _S_, вывести всю строку.

$$K = RIGHT(S, N)$$

## ограничения

* 0 <= |S| <= 1000
* 0 <= N <= 1000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT                 | OUTPUT |
| --------------------- | ------ |
