# Длина строки

Дана строка _S_. Выведите длину данной строки.

## входные данные

В первой строке дана строка символов.

## выходные данные

В одной строке вывести длину строки _S_.

## ограничения

* 0 <= |S| <= 1000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT                 | OUTPUT |
| --------------------- | ------ |
| combination           | 11     |
| this is a test        | 14     |
