# Простое число

Дано число, проверить, является ли оно простым.

## входные данные

В первой строке дано натуральное число __N__.

## выходные данные

Вывести `true` если число __N__ является простым, `false` - в обратном случае. 

## ограничения

 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 6 | false |
| 7 | true |
