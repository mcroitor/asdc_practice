# Сумма произведений

Для заданного натурального числа __n__ найдите сумму $1\times2 + 2\times3 + ... + (n-1)\times n$.

## входные данные

В первой строке дано натуральное число __n__.

## выходные данные

Вывести сумму произведений соседних чисел от 1 до __n__.

## ограничения

* 1 <= n <= 1 000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3     | 8      |
| 5     | 40     |
