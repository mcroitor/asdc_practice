# Сумма квадратов

Для заданного натурального числа __N__ найдите сумму $1^2 + 2^2 + ... +n^2$.

## входные данные

В первой строке дан символ __С__.

## выходные данные

Вывести `true` если __С__ является цифрой, `false` - в обратном случае. 

## ограничения

 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 6 | true |
| с | false |
