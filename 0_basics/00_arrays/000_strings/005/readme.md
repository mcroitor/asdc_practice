# Преобразование регистра символов

Дана строка _S_, состоящая из символов латинского алфавита. Выведите строку _S_, преобразованную к нижнему регистру.

## входные данные

В первой строке дана строка символов.

## выходные данные

В одной строке вывести строку _S_, преобразованную к нижнему регистру.

## ограничения

* 0 <= |S| <= 1000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT       | OUTPUT      |
| ----------- | ----------- |
| CoMBination | combination |
| Hello World | hello world |
