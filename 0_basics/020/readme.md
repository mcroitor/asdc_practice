# Четные числа в диапазоне

По данным числам __a__ и __b__ выведите на экран все четные числа от __a__ до __b__ включительно.

## входные данные

В первой строке даны два целых числа __a__ и __b__, разделенных пробелом.

## выходные данные

Вывести все четные числа от __a__ до __b__ включительно, разделенные пробелом.

## ограничения

* -1000 <= a, b <= 1000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT     |
| ----- | ---------- |
| 1 10  | 2 4 6 8 10 |
| 3 7   | 4 6        |
