# Точные квадраты

Напечатайте все точные квадраты натуральных чисел, не превосходящие данного числа __n__.

## входные данные

В первой строке дано натуральное число __n__.

## выходные данные

Вывести все точные квадраты натуральных чисел, не превосходящие __n__, разделенные пробелом.

## ограничения

 * 1 <= n <= 1 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 50 | 1 4 9 16 25 36 49 |
| 10 | 1 4 9 |
| 100 | 1 4 9 16 25 36 49 64 81 100 |
