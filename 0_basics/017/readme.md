# Сумма цифр числа

Дано натуральное число __n__. Найдите сумму его цифр.

## входные данные

В первой строке дано натуральное число __n__.

## выходные данные

Вывести сумму цифр числа. 

## ограничения

 * n <= 1 000 000 000 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 6 | 6 |
| 12003 | 6 |
