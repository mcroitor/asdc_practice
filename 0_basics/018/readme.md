# Сумма кубов

Для заданного натурального числа __n__ найдите сумму $1^3 + 2^3 + ... + n^3$.

## входные данные

В первой строке дано натуральное число __n__.

## выходные данные

Вывести сумму кубов чисел от 1 до __n__.

## ограничения

* 0 <= n <= 1 000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3     | 36     |
| 5     | 225    |
