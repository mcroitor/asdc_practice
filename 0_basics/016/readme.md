# Четное число

Проверить четность заданного числа.

## входные данные

В первой строке дано натуральное число __N__.

## выходные данные

Вывести `true` если число __N__ является четным, `false` - в обратном случае. 

## ограничения

 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 6 | true |
| 7 | false |
