# реверс

Дан массив из N целых чисел. Выведите эти числа в обратном порядке.

## входные данные

В первой строке целое число N. Во второй строке N целых чисел.

## выходные данные

N введенных целых чисел в обратном порядке.

## ограничения

 * 1 <= N <= 1000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 5<br>-2 -1 0 1 2 | 2 1 0 -1 -2 |
