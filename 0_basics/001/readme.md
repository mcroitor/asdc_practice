# прокси

написать программу, которая считывает число и выводит его же.

## входные данные

В первой строке входных данных задано целое число.

## выходные данные

В первой строке выходных данных вывести целое число, считанное из входных данных.

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 | 10 |
| 2 | 2 |
