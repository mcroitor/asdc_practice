# наибольший общий делитель

Найдите наибольший общий делитель двух натуральных чисел.

## входные данные

Даны два натуральных числа _A_ и _B_, разделенных пробелом.

## выходные данные

Наибольший общий делитель двух натуральных чисел.

## ограничения

 * 0 <= A, B <= 1 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 15 | 5 |
| 27 8 | 1 |
