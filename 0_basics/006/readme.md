# наименьшее общее кратное

Найдите наименьшее общее кратное двух натуральных чисел.

## входные данные

Даны два натуральных числа _A_ и _B_, разделенных пробелом.

## выходные данные

Наименьшее общее кратное двух натуральных чисел.

## ограничения

 * 0 <= A, B <= 1 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 15 | 30 |
| 27 8 | 216 |
