# Цифра

Проверить, является ли вводимый символ цифрой.

## входные данные

В первой строке дан символ __С__.

## выходные данные

Вывести `true` если __С__ является цифрой, `false` - в обратном случае. 

## ограничения

 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 6 | true |
| с | false |
