# Сумма обратных квадратов

По данному числу __n__ вычислите сумму $1+\frac{1}{2^2}+\frac{1}{3^2}+...+\frac{1}{n^2}$.

## входные данные

В первой строке дано натуральное число __n__.

## выходные данные

Вывести значение суммы с точностью до 6 знаков после запятой.

## ограничения

 * 1 <= n <= 100 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 1 | 1.000000 |
| 10 | 1.549768 |
| 100 | 1.634984 |

## примечание

При увеличении __n__ эта сумма приближается к $\frac{\pi^2}{6} \approx 1.644934$.
