# Число сочетаний

По данным натуральным __n__ и __k__ вычислите значение $C_{n}^{k}=\frac{n!}{k!(n-k)!}$ (число сочетаний из __n__ элементов по __k__).

## входные данные

В первой строке даны два натуральных числа __n__ и __k__, разделенных пробелом.

## выходные данные

Вывести значение числа сочетаний $C_{n}^{k}$.

## ограничения

 * 0 <= k <= n <= 20
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 5 2 | 10 |
| 10 3 | 120 |
| 4 4 | 1 |
