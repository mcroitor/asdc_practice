# Факториал

По данному числу __n__ вычислите значение $n!$ (факториал).

## входные данные

В первой строке дано натуральное число __n__.

## выходные данные

Вывести значение $n!$ (произведение всех натуральных чисел от 1 до __n__).

## ограничения

 * 0 <= n <= 20
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 5 | 120 |
| 0 | 1 |
| 10 | 3628800 |
