# Количество пар с простым НОД

Дан массив из N чисел. Найти количество пар элементов (i, j) где i < j, таких что НОД(a[i], a[j]) является простым числом.

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^5). Во второй строке даны __N__ натуральных чисел __a[i]__ (2 ≤ a[i] ≤ 10^6).

## выходные данные

Вывести количество пар с простым НОД.

## ограничения

* 2 ≤ N ≤ 10^5
* 2 ≤ a[i] ≤ 10^6
* time limit: 2s
* memory limit: 32m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 4<br>6 10 15 21 | 4 |
| 3<br>2 4 8 | 3 |
