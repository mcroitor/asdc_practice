# НОД чисел Фибоначчи

Дана последовательность Фибоначчи: F(0) = 0, F(1) = 1, F(n) = F(n-1) + F(n-2).

Известно свойство: НОД(F(n), F(m)) = F(НОД(n, m)).

Даны два натуральных числа _N_ и _M_. Найдите НОД(F(N), F(M)).

## входные данные

Даны два натуральных числа _N_ и _M_, разделенных пробелом.

## выходные данные

НОД чисел Фибоначчи F(N) и F(M).

## ограничения

- 1 <= N, M <= 1 000 000
- time limit: 1s
- memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 6 9   | 2      |
| 10 15 | 5      |
| 12 18 | 8      |

## примечание

Для первого примера:

- F(6) = 8, F(9) = 34
- НОД(6, 9) = 3, F(3) = 2
- Проверка: НОД(8, 34) = 2 ✓

Для второго примера:

- F(10) = 55, F(15) = 610
- НОД(10, 15) = 5, F(5) = 5
- Проверка: НОД(55, 610) = 5 ✓

Для третьего примера:

- F(12) = 144, F(18) = 2584
- НОД(12, 18) = 6, F(6) = 8
- Проверка: НОД(144, 2584) = 8 ✓

**Эффективное решение:** вычислить d = НОД(N, M), затем найти F(d), избегая вычисления огромных чисел Фибоначчи.

Для больших N нужно использовать быстрое возведение матрицы в степень или метод удвоения для вычисления F(d) за O(log d).
