# Взаимно простые числа

Даны два натуральных числа _A_ и _B_. Определите, являются ли они взаимно простыми.

## входные данные

Даны два натуральных числа _A_ и _B_, разделенных пробелом.

## выходные данные

Вывести `YES`, если числа взаимно простые, и `NO` в противном случае.

## ограничения

* 1 <= A, B <= 1 000 000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 15 28 | YES |
| 12 18 | NO |
| 7 13 | YES |
| 100 50 | NO |

## примечание

Два числа называются взаимно простыми, если их наибольший общий делитель равен 1.

В первом примере: НОД(15, 28) = 1, значит числа взаимно простые.

Во втором примере: НОД(12, 18) = 6 ≠ 1, значит числа не взаимно простые.
