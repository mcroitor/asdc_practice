# Постройка египетской пирамиды - 2

Фараон Тутмос III приказал построить новую пирамиду, потому что старая уже не соответствовала его величию. Для строительства в мраморном карьере был вырублен огромный блок камня в форме прямоугольного параллелепипеда с длинами рёбер _A_, _B_ и _C_. Чтобы упростить себе (и каменотесам) работу, главный архитектор Аменка решил разрезать этот блок на кирпичи в виде куба одинаковых размеров, не оставляя при этом остатков. Это позволит сделать все кирпичи одинаковыми и выполнить наименьшее количество разрезов. Помогите архитектору определить, какое минимальное количество распилов потребуется для получения одинаковых кирпичей (в виде кубов) из исходного блока.

## входные данные

Даны три натуральных числа _A_, _B_ и _C_, разделенных пробелом — длины рёбер прямоугольного параллелепипеда.

## выходные данные

Два числа, разделенных пробелом: максимальная длина ребра куба и минимальное количество распилов, необходимых для получения одинаковых кубов из исходного блока.

## ограничения

* 1 <= A, B, C <= 1 000 000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT       | OUTPUT |
| ----------- | ------ |
| 1 2 3       | 1 3    |
| 15 25 35    | 5 12   |

## примечание

Максимальный размер куба равен НОД длин рёбер прямоугольного параллелепипеда. Количество разрезов можно найти по формуле: (A ÷ НОД(A, B, C) - 1) + (B ÷ НОД(A, B, C) - 1) + (C ÷ НОД(A, B, C) - 1).

В первом примере: НОД(1, 2, 3) = 1. Прямоугольный параллелепипед 1×2×3 можно разрезать на кубы 1×1×1. Потребуется (1÷1 - 1) + (2÷1 - 1) + (3÷1 - 1) = 0 + 1 + 2 = 3 разреза.

Во втором примере: НОД(15, 25, 35) = 5. Прямоугольный параллелепипед 15×25×35 можно разрезать на кубы 5×5×5. Потребуется (15÷5 - 1) + (25÷5 - 1) + (35÷5 - 1) = 2 + 4 + 6 = 12 разрезов.
