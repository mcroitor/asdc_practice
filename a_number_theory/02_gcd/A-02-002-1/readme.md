# НОД трех чисел

Найдите наибольший общий делитель трех натуральных чисел.

## входные данные

Даны три натуральных числа _A_, _B_ и _C_, разделенных пробелом.

## выходные данные

Наибольший общий делитель трех натуральных чисел.

## ограничения

 * 1 <= A, B, C <= 1 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 18 24 | 6 |
| 15 25 35 | 5 |
| 7 11 13 | 1 |

## примечание

НОД(a, b, c) = НОД(НОД(a, b), c)
