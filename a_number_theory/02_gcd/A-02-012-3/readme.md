# Сумма НОД всех пар

Дано натуральное число _N_. Вычислите сумму НОД всех пар чисел от 1 до N:

$$\sum_{i=1}^{N} \sum_{j=1}^{N} \text{НОД}(i, j)$$

## входные данные

Дано натуральное число _N_.

## выходные данные

Сумма НОД всех пар чисел от 1 до N.

## ограничения

* 1 <= N <= 100 000
* time limit: 2s
* memory limit: 64m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3     | 12     |
| 5     | 37     |
| 10    | 182    |

## примечание

Для N = 3:

```text
i\j  1  2  3
1    1  1  1  → 3
2    1  2  1  → 4
3    1  1  3  → 5
Итого: 3 + 4 + 5 = 12
```

Для N = 5:

```text
i=1: 1+1+1+1+1 = 5
i=2: 1+2+1+2+1 = 7
i=3: 1+1+3+1+1 = 7
i=4: 1+2+1+4+1 = 9
i=5: 1+1+1+1+5 = 9
Итого: 5+7+7+9+9 = 37
```

**Наивное решение O(N²)** будет слишком медленным для N = 100000.

**Эффективное решение** использует формулу:

$$\sum_{i=1}^{N} \sum_{j=1}^{N} \text{НОД}(i, j) = \sum_{d=1}^{N} d \cdot \phi(d) \cdot \left\lfloor \frac{N}{d} \right\rfloor^2 + \sum_{d=1}^{N} d \cdot \left( 2 \cdot \sum_{k=1}^{\lfloor N/d \rfloor - 1} \phi(k) \right) \cdot \left\lfloor \frac{N}{d} \right\rfloor$$

где φ(k) — функция Эйлера (количество чисел от 1 до k, взаимно простых с k).

Более простая формула:
$$\sum_{i=1}^{N} \sum_{j=1}^{N} \text{НОД}(i, j) = 2 \cdot \sum_{i=1}^{N} \sum_{j=1}^{i-1} \text{НОД}(i, j) + \sum_{i=1}^{N} i$$

Это классическая олимпиадная задача, требующая знания теории чисел и оптимизации.
