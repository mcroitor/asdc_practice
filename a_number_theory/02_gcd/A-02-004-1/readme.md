# Сокращение дроби

Дана дробь _A/B_. Сократите дробь до несократимого вида.

## входные данные

Даны два натуральных числа _A_ и _B_, разделенных пробелом, представляющие дробь A/B.

## выходные данные

Два числа, разделенных пробелом: числитель и знаменатель несократимой дроби.

## ограничения

 * 1 <= A, B <= 1 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 18 | 2 3 |
| 15 25 | 3 5 |
| 7 13 | 7 13 |
| 100 150 | 2 3 |

## примечание

Для сокращения дроби нужно разделить числитель и знаменатель на их НОД.

Несократимая дробь — это дробь, у которой числитель и знаменатель взаимно простые (их НОД равен 1).

В первом примере: НОД(12, 18) = 6, поэтому 12/18 = (12÷6)/(18÷6) = 2/3.
