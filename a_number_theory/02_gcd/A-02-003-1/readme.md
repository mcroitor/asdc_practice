# НОД массива чисел

Дан массив из _N_ натуральных чисел. Найдите наибольший общий делитель всех элементов массива.

## входные данные

В первой строке дано натуральное число _N_ — размер массива.
Во второй строке даны _N_ натуральных чисел, разделенных пробелом.

## выходные данные

Наибольший общий делитель всех элементов массива.

## ограничения

 * 1 <= N <= 1000
 * 1 <= A[i] <= 1 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3<br>12 18 24 | 6 |
| 5<br>10 20 30 40 50 | 10 |
| 4<br>7 14 21 28 | 7 |

## примечание

НОД последовательно вычисляется для каждого элемента: НОД(a₁, a₂, ..., aₙ) = НОД(...НОД(НОД(a₁, a₂), a₃)..., aₙ)
