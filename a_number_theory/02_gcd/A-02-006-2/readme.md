# Максимальный квадрат

Дан прямоугольник с целочисленными сторонами _A_ и _B_. Требуется разрезать его на максимальные квадраты с целочисленной стороной так, чтобы не осталось отходов. Нужно определить сторону такого квадрата и количество квадратов.

## входные данные

Даны два натуральных числа _A_ и _B_, разделенных пробелом — длины сторон прямоугольника.

## выходные данные

Длина стороны максимального квадрата, на которые можно разрезать прямоугольник без остатка, и количество таких квадратов.

## ограничения

* 1 <= A, B <= 1 000 000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT       | OUTPUT |
| ----------- | ------ |
| 1 2         | 1 2    |
| 15 25       | 5 15   |
| 7 13        | 1 91   |
| 100 150     | 50 6   |

## примечание

Максимальный размер квадрата равен НОД длин сторон прямоугольника. Количество квадратов вычисляется как (A ÷ НОД(A, B)) × (B ÷ НОД(A, B)).

В первом примере: НОД(1, 2) = 1. Прямоугольник 1×2 можно разрезать на квадраты 1×1. Потребуется (1÷1) × (2÷1) = 1 × 2 = 2 квадрата.

Во втором примере: НОД(15, 25) = 5. Прямоугольник 15×25 можно разрезать на квадраты 5×5. Потребуется (15÷5) × (25÷5) = 3 × 5 = 15 квадратов.
