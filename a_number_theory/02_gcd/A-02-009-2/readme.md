# Бинарный НОД

Реализуйте бинарный алгоритм Евклида для вычисления НОД двух натуральных чисел. Этот алгоритм использует только операции сравнения, деления и умножения на 2 (битовые сдвиги).

## входные данные

Даны два натуральных числа _A_ и _B_, разделенных пробелом.

## выходные данные

Наибольший общий делитель чисел _A_ и _B_.

## ограничения

 * 1 <= A, B <= 10^18
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 18 | 6 |
| 15 25 | 5 |
| 1000000000000 500000000000 | 500000000000 |

## примечание

Бинарный алгоритм Евклида (алгоритм Штейна):

1. Если A = B, то НОД(A, B) = A
2. Если A = 0, то НОД(A, B) = B
3. Если B = 0, то НОД(A, B) = A
4. Если A и B четные, то НОД(A, B) = 2 × НОД(A/2, B/2)
5. Если A четное, B нечетное, то НОД(A, B) = НОД(A/2, B)
6. Если A нечетное, B четное, то НОД(A, B) = НОД(A, B/2)
7. Если A и B нечетные и A > B, то НОД(A, B) = НОД((A-B)/2, B)
8. Если A и B нечетные и A < B, то НОД(A, B) = НОД(A, (B-A)/2)

Преимущество: не использует операцию деления с остатком, работает быстрее на больших числах.

Задача проверяет понимание альтернативных алгоритмов вычисления НОД и работу с большими числами.
