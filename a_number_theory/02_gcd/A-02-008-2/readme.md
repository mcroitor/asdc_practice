# Количество делителей НОД

Даны два натуральных числа _A_ и _B_. Найдите количество делителей их наибольшего общего делителя.

## входные данные

Даны два натуральных числа _A_ и _B_, разделенных пробелом.

## выходные данные

Количество делителей НОД(A, B).

## ограничения

 * 1 <= A, B <= 1 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 18 | 4 |
| 15 25 | 2 |
| 100 150 | 6 |
| 7 13 | 1 |

## примечание

Для первого примера:
- НОД(12, 18) = 6
- Делители числа 6: 1, 2, 3, 6
- Количество делителей: 4

Для второго примера:
- НОД(15, 25) = 5
- Делители числа 5: 1, 5
- Количество делителей: 2

Для третьего примера:
- НОД(100, 150) = 50
- Делители числа 50: 1, 2, 5, 10, 25, 50
- Количество делителей: 6

Подсказка: после нахождения НОД нужно найти все его делители. Для эффективного решения можно проверять делители до √НОД.
