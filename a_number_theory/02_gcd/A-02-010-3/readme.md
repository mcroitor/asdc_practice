# Расширенный алгоритм Евклида

Даны два натуральных числа _A_ и _B_. Найдите такие целые числа _x_ и _y_, что _A × x + B × y = НОД(A, B)_.

## входные данные

Даны два натуральных числа _A_ и _B_, разделенных пробелом.

## выходные данные

Три числа, разделенных пробелом: НОД(A, B), затем коэффициенты _x_ и _y_.

## ограничения

* 1 <= A, B <= 1 000 000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 18 | 6 -1 1 |
| 15 25 | 5 2 -1 |
| 7 13 | 1 2 -1 |

## примечание

Для первого примера: НОД(12, 18) = 6
Проверка: 12 × (-1) + 18 × 1 = -12 + 18 = 6 ✓

Для второго примера: НОД(15, 25) = 5
Проверка: 15 × 2 + 25 × (-1) = 30 - 25 = 5 ✓

Расширенный алгоритм Евклида:

```text
функция extGCD(a, b):
    если b = 0:
        вернуть (a, 1, 0)
    иначе:
        (d, x₁, y₁) = extGCD(b, a mod b)
        x = y₁
        y = x₁ - (a div b) × y₁
        вернуть (d, x, y)
```

Решение не единственное. Если (x₀, y₀) — решение, то (x₀ + k×(B/d), y₀ - k×(A/d)) тоже решение для любого целого k.

Расширенный алгоритм Евклида используется для решения диофантовых уравнений и в криптографии.
