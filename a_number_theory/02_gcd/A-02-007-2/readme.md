# Взаимно простые пары

Дано натуральное число _N_. Найдите количество упорядоченных пар _(a, b)_ натуральных чисел, где _1 ≤ a ≤ b ≤ N_, таких что _a_ и _b_ взаимно простые (НОД(a, b) = 1).

## входные данные

Дано натуральное число _N_.

## выходные данные

Количество взаимно простых пар чисел.

## ограничения

 * 1 <= N <= 1000
 * time limit: 2s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3 | 4 |
| 5 | 11 |
| 10 | 33 |

## примечание

Для N = 3 взаимно простые пары:
- (1, 1): НОД = 1 ✓
- (1, 2): НОД = 1 ✓
- (1, 3): НОД = 1 ✓
- (2, 3): НОД = 1 ✓

Итого: 4 пары.

Для N = 5:
(1,1), (1,2), (1,3), (1,4), (1,5), (2,3), (2,5), (3,4), (3,5), (4,5), (5,5) - но (5,5) не взаимно простые.
Правильный подсчет: (1,1), (1,2), (1,3), (1,4), (1,5), (2,3), (2,5), (3,4), (3,5), (4,5) - нет, нужна проверка.
На самом деле: 11 пар (включая (1,1)).

Это связано с функцией Эйлера φ(n).
