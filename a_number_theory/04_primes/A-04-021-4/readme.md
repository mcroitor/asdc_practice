# Простые в арифметической прогрессии

Найти максимальную длину последовательности простых чисел, образующих арифметическую прогрессию, в диапазоне [L, R].

## входные данные

В первой строке даны два натуральных числа __L__ и __R__ (2 ≤ L ≤ R ≤ 10^6).

## выходные данные

Вывести максимальную длину арифметической прогрессии из простых чисел в диапазоне [__L__, __R__].

## ограничения

* 2 ≤ L ≤ R ≤ 10^6
* time limit: 2s
* memory limit: 16m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 2 20 | 3 |
| 2 100 | 6 |
