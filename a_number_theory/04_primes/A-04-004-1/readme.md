# Количество простых чисел до N

Подсчитать количество простых чисел, не превышающих N.

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^6).

## выходные данные

Вывести количество простых чисел от 2 до __N__ включительно.

## ограничения

* 2 ≤ N ≤ 10^6
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 | 4 |
| 100 | 25 |
| 1000 | 168 |
