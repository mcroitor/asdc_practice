# Минимальное количество простых в сумме

Найти минимальное количество простых чисел, сумма которых равна N.

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^6).

## выходные данные

Вывести минимальное количество простых чисел, сумма которых равна __N__. Простые числа могут повторяться.

## ограничения

* 2 ≤ N ≤ 10^6
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 7 | 1 |
| 10 | 2 |
| 13 | 1 |
| 27 | 3 |
