# Разложение на простые множители

Выполнить полную факторизацию числа на простые множители.

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^9).

## выходные данные

Вывести разложение числа __N__ на простые множители в формате `p1^k1 * p2^k2 * ... * pm^km`, где p1 < p2 < ... < pm - простые делители, ki - их степени. Если степень равна 1, показатель степени не выводить.

## ограничения

* 2 ≤ N ≤ 10^9
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 | 2^2 * 3 |
| 100 | 2^2 * 5^2 |
| 17 | 17 |
