# Числа-близнецы

Найти все пары простых чисел-близнецов в диапазоне [L, R]. Близнецы - это пары простых чисел (p, p+2).

## входные данные

В первой строке даны два натуральных числа __L__ и __R__ (2 ≤ L ≤ R ≤ 10^6).

## выходные данные

Вывести количество пар простых чисел-близнецов, где оба числа находятся в диапазоне [__L__, __R__].

## ограничения

* 2 ≤ L ≤ R ≤ 10^6
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 2 20 | 4 |
| 100 200 | 2 |
