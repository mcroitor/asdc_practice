# Ближайшее простое число

Найти ближайшее простое число к данному N (не меньшее N).

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^6).

## выходные данные

Вывести наименьшее простое число, которое больше или равно __N__.

## ограничения

* 2 ≤ N ≤ 10^6
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 | 11 |
| 14 | 17 |
| 17 | 17 |
