# Числа Софи Жермен

Найти все числа Софи Жермен в диапазоне [L, R]. Число p - это число Софи Жермен, если p простое и 2p+1 также простое.

## входные данные

В первой строке даны два натуральных числа __L__ и __R__ (2 ≤ L ≤ R ≤ 10^6).

## выходные данные

Вывести количество чисел Софи Жермен в диапазоне [__L__, __R__].

## ограничения

* 2 ≤ L ≤ R ≤ 10^6
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 2 20 | 4 |
| 100 200 | 6 |
