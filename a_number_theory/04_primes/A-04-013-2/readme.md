# Простые палиндромы

Найти все простые числа-палиндромы в диапазоне [L, R].

## входные данные

В первой строке даны два натуральных числа __L__ и __R__ (2 ≤ L ≤ R ≤ 10^6).

## выходные данные

Вывести количество простых чисел-палиндромов в диапазоне [__L__, __R__].

## ограничения

* 2 ≤ L ≤ R ≤ 10^6
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 2 20 | 4 |
| 100 200 | 2 |
| 1 1000 | 20 |
