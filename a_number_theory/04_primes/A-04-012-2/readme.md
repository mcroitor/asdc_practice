# Суперпростые числа

Найти все суперпростые числа в диапазоне [L, R] - числа, у которых все цифры простые И само число простое.

## входные данные

В первой строке даны два натуральных числа __L__ и __R__ (2 ≤ L ≤ R ≤ 10^6).

## выходные данные

Вывести все суперпростые числа в диапазоне [__L__, __R__] через пробел в порядке возрастания. Если таких чисел нет, вывести `-1`.

## ограничения

* 2 ≤ L ≤ R ≤ 10^6
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 20 30 | 23 |
| 2 10 | 2 3 5 7 |
| 200 210 | -1 |
