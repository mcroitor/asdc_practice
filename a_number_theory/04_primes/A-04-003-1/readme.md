# N-е простое число

Найти N-е простое число по порядку.

## входные данные

В первой строке дано натуральное число __N__ (1 ≤ N ≤ 10000).

## выходные данные

Вывести __N__-е простое число (первое простое число - это 2).

## ограничения

* 1 ≤ N ≤ 10000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 1 | 2 |
| 10 | 29 |
| 100 | 541 |
