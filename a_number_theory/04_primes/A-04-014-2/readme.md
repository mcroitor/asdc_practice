# Сегментированное решето

Найти все простые числа в диапазоне [L, R] при больших значениях L и R.

## входные данные

В первой строке даны два натуральных числа __L__ и __R__ (2 ≤ L ≤ R ≤ 10^12, R - L ≤ 10^6).

## выходные данные

Вывести количество простых чисел в диапазоне [__L__, __R__].

## ограничения

* 2 ≤ L ≤ R ≤ 10^12
* R - L ≤ 10^6
* time limit: 2s
* memory limit: 32m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 20 | 4 |
| 1000000000000 1000000001000 | 37 |
