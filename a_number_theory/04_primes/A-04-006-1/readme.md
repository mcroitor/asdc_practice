# Решето Эратосфена

Реализовать алгоритм решета Эратосфена для нахождения всех простых чисел до N.

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^7).

## выходные данные

Вывести количество простых чисел от 2 до __N__ включительно.

## ограничения

* 2 ≤ N ≤ 10^7
* time limit: 2s
* memory limit: 64m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 | 4 |
| 1000000 | 78498 |
