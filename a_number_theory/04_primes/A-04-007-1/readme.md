# Простые делители числа

Найти все простые делители данного числа.

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^9).

## выходные данные

Вывести все различные простые делители числа __N__ в порядке возрастания через пробел.

## ограничения

* 2 ≤ N ≤ 10^9
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 | 2 3 |
| 30 | 2 3 5 |
| 17 | 17 |
