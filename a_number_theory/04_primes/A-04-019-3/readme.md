# Гипотеза Гольдбаха

Для данного четного числа N > 2, найти два простых числа, сумма которых равна N (слабая гипотеза Гольдбаха).

## входные данные

В первой строке дано четное натуральное число __N__ (4 ≤ N ≤ 10^6, N - четное).

## выходные данные

Вывести два простых числа __p1__ и __p2__ (p1 ≤ p2), сумма которых равна __N__. Если существует несколько решений, вывести пару с минимальным __p1__.

## ограничения

* 4 ≤ N ≤ 10^6
* N - четное
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 | 3 7 |
| 100 | 3 97 |
| 1000 | 3 997 |
