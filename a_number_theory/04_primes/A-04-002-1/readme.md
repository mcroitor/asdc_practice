# Простые числа в диапазоне

Найти все простые числа в диапазоне от 1 до N.

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^6).

## выходные данные

Вывести все простые числа от 1 до __N__ в порядке возрастания через пробел.

## ограничения

* 2 ≤ N ≤ 10^6
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 10 | 2 3 5 7 |
| 20 | 2 3 5 7 11 13 17 19 |
