# НОК трех чисел

Найдите наименьшее общее кратное трех натуральных чисел.

## входные данные

Даны три натуральных числа _A_, _B_ и _C_, разделенных пробелом.

## выходные данные

Наименьшее общее кратное трех натуральных чисел.

## ограничения

 * 1 <= A, B, C <= 1 000 000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 4 6 8 | 24 |
| 5 10 15 | 30 |
| 7 11 13 | 1001 |

## примечание

НОК(a, b, c) = НОК(НОК(a, b), c)
