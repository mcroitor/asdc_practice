# Синхронизация светофоров

На перекрестке установлены два светофора. Первый светофор горит зеленым _A_ секунд, затем красным _B_ секунд, и так циклически. Второй светофор горит зеленым _C_ секунд, затем красным _D_ секунд. В момент времени 0 оба светофора начинают гореть зеленым одновременно.

Через сколько секунд после начала оба светофора снова одновременно загорятся зеленым?

## входные данные

В одной строке даны четыре натуральных числа _A_, _B_, _C_ и _D_, разделенных пробелом.

## выходные данные

Количество секунд до следующей синхронизации (момента, когда оба светофора снова одновременно начнут гореть зеленым).

## ограничения

 * 1 <= A, B, C, D <= 1000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3 2 4 3 | 35 |
| 5 5 6 4 | 10 |
| 10 10 15 15 | 60 |

## примечание

Период первого светофора: A + B секунд.
Период второго светофора: C + D секунд.
Ответ: НОК(A + B, C + D) секунд.

В первом примере: период₁ = 5, период₂ = 7, НОК(5, 7) = 35.
