# НОК чисел от 1 до N

Найдите наименьшее общее кратное всех натуральных чисел от 1 до _N_.

## входные данные

Дано натуральное число _N_.

## выходные данные

НОК всех чисел от 1 до _N_.

## ограничения

 * 1 <= N <= 20
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3 | 6 |
| 4 | 12 |
| 5 | 60 |
| 10 | 2520 |

## примечание

НОК(1, 2, 3) = 6
НОК(1, 2, 3, 4) = 12
НОК(1, 2, 3, 4, 5) = 60

Для эффективного вычисления можно использовать факторизацию: НОК содержит каждый простой множитель в максимальной степени, встречающейся в числах от 1 до N.

Например, для N=10: НОК = 2³ × 3² × 5 × 7 = 2520
