# Пары с заданным НОК

Дано число _L_. Найдите количество упорядоченных пар натуральных чисел _(a, b)_ таких, что _a ≤ b ≤ L_ и НОК(a, b) = L.

## входные данные

Дано натуральное число _L_.

## выходные данные

Количество упорядоченных пар _(a, b)_, где _a ≤ b ≤ L_ и НОК(a, b) = L.

## ограничения

 * 1 <= L <= 1 000 000
 * time limit: 2s
 * memory limit: 16m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 6 | 5 |
| 12 | 13 |
| 1 | 1 |
| 10 | 5 |

## примечание

Для L = 6 подходят пары: (1,6), (2,6), (3,6), (6,6), (5,6) - нет, НОК(5,6)=30.
На самом деле: (1,6), (2,6), (3,6), (6,6) - это 4 пары. Нужно пересчитать.

Правильный подсчет для L = 6:
- (1, 6): НОК = 6 ✓
- (2, 6): НОК = 6 ✓
- (3, 6): НОК = 6 ✓
- (6, 6): НОК = 6 ✓
- (2, 3): НОК = 6 ✓

Итого: 5 пар.

Подсказка: пара (a, b) имеет НОК = L тогда и только тогда, когда оба числа являются делителями L, и хотя бы одно из них равно L.
