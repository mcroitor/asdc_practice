# Числа с заданными НОК и НОД

Даны два натуральных числа _G_ и _L_. Найдите количество упорядоченных пар натуральных чисел _(a, b)_ таких, что НОД(a, b) = G и НОК(a, b) = L.

## входные данные

Даны два натуральных числа _G_ и _L_, разделенных пробелом.

## выходные данные

Количество упорядоченных пар _(a, b)_, удовлетворяющих условиям, или 0, если таких пар не существует.

## ограничения

* 1 <= G, L <= 1 000 000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 2 12  | 4      |
| 3 15  | 2      |
| 4 10  | 0      |
| 1 1   | 1      |

## примечание

Необходимое условие существования решения: G должен делить L, иначе ответ 0.

Для G = 2, L = 12:

* Пары (a, b) должны иметь НОД = 2 и НОК = 12
* Возможные пары: (2, 12), (12, 2), (4, 6), (6, 4)
* Проверка: НОД(2, 12) = 2, НОК(2, 12) = 12 ✓
* Проверка: НОД(4, 6) = 2, НОК(4, 6) = 12 ✓
* Всего: 4 пары

Для G = 4, L = 10:

* L не делится на G, значит решений нет.

Подсказка: используйте формулу a × b = НОД(a, b) × НОК(a, b). Если положить a = G × x и b = G × y, где НОД(x, y) = 1, то НОК(a, b) = G × НОК(x, y) = G × x × y = L.
