# НОК массива чисел

Дан массив из _N_ натуральных чисел. Найдите наименьшее общее кратное всех элементов массива.

## входные данные

В первой строке дано натуральное число _N_ — размер массива.
Во второй строке даны _N_ натуральных чисел, разделенных пробелом.

## выходные данные

Наименьшее общее кратное всех элементов массива.

## ограничения

 * 1 <= N <= 100
 * 1 <= A[i] <= 1000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3<br>4 6 8 | 24 |
| 5<br>2 3 4 5 6 | 60 |
| 4<br>7 14 21 28 | 84 |

## примечание

НОК последовательно вычисляется для каждого элемента: НОК(a₁, a₂, ..., aₙ) = НОК(...НОК(НОК(a₁, a₂), a₃)..., aₙ)
