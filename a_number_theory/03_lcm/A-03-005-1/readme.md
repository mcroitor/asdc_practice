# Общий знаменатель дробей

Даны _N_ дробей в виде числителя и знаменателя. Найдите наименьший общий знаменатель для всех дробей.

## входные данные

В первой строке дано натуральное число _N_ — количество дробей.
В следующих _N_ строках даны две натуральных числа _A_ и _B_, разделенных пробелом, представляющие дробь A/B.

## выходные данные

Наименьший общий знаменатель для всех дробей (НОК всех знаменателей).

## ограничения

 * 1 <= N <= 50
 * 1 <= A, B <= 1000
 * time limit: 1s
 * memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3<br>1 2<br>1 3<br>1 4 | 12 |
| 2<br>3 5<br>7 10 | 10 |
| 4<br>1 6<br>1 8<br>1 12<br>1 15 | 120 |

## примечание

Для приведения дробей к общему знаменателю нужно найти НОК всех знаменателей.

В первом примере: НОК(2, 3, 4) = 12, что позволяет привести дроби 1/2, 1/3, 1/4 к виду 6/12, 4/12, 3/12.
