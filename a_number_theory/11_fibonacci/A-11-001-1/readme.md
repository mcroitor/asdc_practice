# Числа Фибоначчи

Числами Фибоначчи называется последовательность чисел __F__, удовлетворяющая следующим условиям:

* __F(0) = 0__
* __F(1) = 1__
* __F(N) = F(N - 1) + F(N - 2)__

Найдите число Фибоначчи с индексом _N_.

## входные данные

Дано натуральное число _N_.

## выходные данные

Число Фибоначчи с индексом _N_.

## ограничения

* 0 <= N <= 1 000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 3     | 2      |
| 10    | 55     |
