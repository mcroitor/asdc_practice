# период числа - 2

Даны два натуральных числа $A, B$. Найдите запись дроби $A / B$ в десятичном виде.

## входные данные

В первой строке даны два натуральных числа $A$ и $B$, разделенные пробелом.

## выходные данные

Выведите дробь $A / B$, записанную в десятичном виде.

## ограничения

* 1 <= A, B <= 1000000
* time limit: 1s
* memory limit: 8m

## пример

| INPUT  | OUTPUT  |
| ------ | ------- |
| 1 10   | 0.1     |
| 1 3    | 0.(3)   |
| 121 36 | 3.36(1) |
| 5 1    | 5       |
