# Количество делителей числа

Используя факторизацию, найти количество всех делителей числа N.

## входные данные

В первой строке дано натуральное число __N__ (1 ≤ N ≤ 10^9).

## выходные данные

Вывести количество всех делителей числа __N__ (включая 1 и само число).

## ограничения

* 1 ≤ N ≤ 10^9
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 | 6 |
| 17 | 2 |
| 24 | 8 |
