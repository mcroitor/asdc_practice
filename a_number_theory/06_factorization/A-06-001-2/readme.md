# Сумма простых делителей

Найти сумму всех различных простых делителей числа N.

## входные данные

В первой строке дано натуральное число __N__ (2 ≤ N ≤ 10^9).

## выходные данные

Вывести сумму всех различных простых делителей числа __N__.

## ограничения

* 2 ≤ N ≤ 10^9
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 12 | 5 |
| 30 | 10 |
| 17 | 17 |
