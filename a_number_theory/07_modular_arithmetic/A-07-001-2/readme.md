# Функция Эйлера

Вычислить значение функции Эйлера φ(n) - количество чисел от 1 до n, взаимно простых с n.

## входные данные

В первой строке дано натуральное число __N__ (1 ≤ N ≤ 10^9).

## выходные данные

Вывести значение φ(__N__).

## ограничения

* 1 ≤ N ≤ 10^9
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 1 | 1 |
| 5 | 4 |
| 12 | 4 |
