# Теорема Вильсона

Проверить, что (p-1)! ≡ -1 (mod p) для данного простого числа p (теорема Вильсона).

## входные данные

В первой строке дано простое число __p__ (2 ≤ p ≤ 10^4).

## выходные данные

Вывести значение (p-1)! mod p.

## ограничения

* 2 ≤ p ≤ 10^4
* p - простое число
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 5 | 4 |
| 7 | 6 |
| 11 | 10 |
