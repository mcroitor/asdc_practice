# Малая теорема Ферма

Вычислить a^(p-1) mod p для простого числа p и проверить, что результат равен 1 (малая теорема Ферма).

## входные данные

В первой строке даны два натуральных числа __a__ и __p__ (2 ≤ a < p ≤ 10^9, p - простое).

## выходные данные

Вывести `true` если a^(p-1) ≡ 1 (mod p), `false` - в обратном случае.

## ограничения

* 2 ≤ a < p ≤ 10^9
* p - простое число
* time limit: 1s
* memory limit: 8m

## пример

| INPUT | OUTPUT |
| ----- | ------ |
| 2 7 | true |
| 3 11 | true |
| 5 13 | true |
