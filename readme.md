# Задачи по программированию для тренировки

Коллекция задач по программированию для самостоятельной тренировки и подготовки к соревнованиям. Задачи разбиты на темы и уровни сложности, согласно [силлабусу](./syllabus.md).

## Код задачи

Каждая задача имеет уникальный код, имеющий следующий формат:

```text
<ОБЛАСТЬ>-<ТЕМА>-<НОМЕР_ЗАДАЧИ>-<СЛОЖНОСТЬ>
```

где:

- `<ОБЛАСТЬ>` — буквенный код области согласно силлабусу (например, `A` для Теории чисел, `B` для Структур данных и т.д.);
- `<ТЕМА>` — двухзначный номер темы в области согласно силлабусу (например, `01` для "Делимость целых чисел" в области Теории чисел);
- `<НОМЕР_ЗАДАЧИ>` — порядковый номер задачи в данной теме и уровне сложности, начиная с `001`;
- `<СЛОЖНОСТЬ>` — уровень сложности задачи: `1` (простой), `2` (средний), `3` (сложный), `4` (очень сложный).
